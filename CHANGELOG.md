# Changelog

## [1.0.0] — Initial Release

### Added
- **Fix Entire Project** — scan up to 50 files, auto-detect and fix all issues with diff preview
- **Generate from Intent** — type a goal in plain English, get a working multi-file project
- **Fix & Explain Selection** — select code, get explanation + fixed version side-by-side
- **Refactor Current File** — choose refactoring goals (performance, readability, security, etc.)
- **Generate Tests** — auto-write comprehensive test suite for any file
- Multi-provider support: **Ollama** (local/free), **Gemini** (free tier), **OpenAI-compatible**
- Streaming responses with live activity log in sidebar
- Plan confirmation before execution — see exactly what will be created
- Diff preview before applying bulk changes
- One-click provider configuration wizard
- Keyboard shortcuts: `Ctrl+Shift+F` (fix project), `Ctrl+Shift+G` (generate), `Ctrl+Shift+A` (panel)
- Status bar indicator with live agent activity
